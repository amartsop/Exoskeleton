Class Sensor.

Initialize sensor.

This function initializes the sensor based on the configuration made by ExoHandler::initialize_sensors. It stores a local copy of all the sensor's properties to it's local member variables.

Parameters
sensor_idThe ID of the sensor.
sensor_typeThe sensor type (0 for Red and 1 for Green).
sensor_portThe port in which the sensor is connected.
analog_minThe minimum analog value that the sensor reads (this can be deducted by testing).
analog_maxThe maximum analog value that the sensor reads (this can be deducted by testing).
angle_minThe intended minimum angle that the sensor is supposed to read in degrees.
angle_maxThe intended maximum angle that the sensor is supposed to read in degrees.

◆ update()

void Sensor::update ( const AnalogPort::AnalogPortValues &  port_values)

Update sensor.

The function updates the snesor analog and angle values based on the incoming analog measurements and its color.The analog value is mapped to the angle value with the help of Arduino's built-in "map" function.

Parameters
port_valuesThe values read at the port in which the sensor is connected.
